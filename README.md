## Installation
First, you need to install Ansible :
```
git clone git@github.com:InsolvencyService/LaptopToolKit.git
cd LaptopToolKit
./install-ansible.sh
```

Run ```ansible-playbook ansible-setup.yml --ask-become-pass```
